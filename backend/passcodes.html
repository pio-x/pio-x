Spielregeln

Stationen

Alle 5 Minuten werden Punkte für Stationen die den Teams gehören verteilt.
Stationen einnehmen gibt keine Punkte.
Stationen dürfen weder von innerhalb eines Busses aus eingenommen werden, noch darf man die 'raus-click-rein' Technik benutzen. Nach dem Einnehmen einer Station muss man den Bus abfahren lassen.

MisterX

Jeder MisterX kann pro Gruppe nur einmal gefangen werden.
Infos über alle MisterX's können immer nach einer gewissen Punkteanzahl angezeigt werden. Dafür kann die Gruppe manuell einen 'Timer' starten. Für eine gewisse Zeit werden dann alle MisterX angezeigt.
MisterX's verlassen ihr vorgegebenes Gebiet nicht.

Rätsel

Ein Rätsel ist nur einmal pro Gruppe lösbar.
Einige Rätsel können erst freigeschaltet werden wenn man in der Nähe ist ('freigeschaltet' = Rätseltext kann angezeigt werden).

Passcodes

Code (und evtl. Bild) eingeben um Punkte zu erhalten
Passcodes werden für Rätsel und MisterX verwendet.
Einige Rätsel können nach einem obligatorischen Bild verlangen.
